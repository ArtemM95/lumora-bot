# Lumora Bot — инструкция по запуску

## Шаг 1 — Вставь токен в bot.py

Открой файл `bot.py`, найди строку:
```
TOKEN = "ВСТАВЬ_ТОКЕН_СЮДА"
```
Замени на свой токен от @BotFather:
```
TOKEN = "1234567890:AABBcc..."
```

---

## Шаг 2 — Запуск на Railway.app (бесплатно)

1. Зайди на https://railway.app
2. Войди через GitHub (создай аккаунт если нет)
3. New Project → Deploy from GitHub repo
4. Загрузи оба файла: bot.py и requirements.txt
5. В настройках проекта → Variables → добавь:
   - KEY: `TOKEN`
   - VALUE: твой токен от BotFather
6. В bot.py замени строку токена на:
   ```python
   TOKEN = os.environ.get("TOKEN")
   ```
7. Deploy — бот запустится автоматически

---

## Шаг 3 — Первый запуск

Найди своего бота в Telegram по имени которое дал @BotFather
Напиши `/start` — бот пришлёт задачи на сегодня

---

## Команды бота

- `/start` или `/today` — задачи на сегодня
- `/stats` — твой прогресс, XP, серия дней
- `/phase` — переключить фазу (месяц 1/2/3)

---

## Локальный запуск на Mac (для теста)

```bash
pip3 install python-telegram-bot==20.7 pytz
python3 bot.py
```

Бот будет работать пока открыт терминал.
Для постоянной работы — деплой на Railway.

---

## Что делает бот

- Каждое утро в 9:00 (Belgrade time) присылает задачи на день
- Ты нажимаешь кнопки — задачи отмечаются
- За каждую задачу начисляется XP
- Считает серию активных дней
- Разовые задачи отмечаются один раз навсегда
- Ежедневные — каждый день заново
